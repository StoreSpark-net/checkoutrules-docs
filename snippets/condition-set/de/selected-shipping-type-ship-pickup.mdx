### Ausgewählte Versandart (Versand/Abholung)

Diese Bedingung prüft die Auswahl „Versand vs. Abholung im Geschäft“ auf der Checkout-Seite, wie unten gezeigt.

<img
  src="/images/essentials/condition-sets/shipping-pickup-type.webp"
  alt="Checkout-Seite Versand vs. Abholung Auswahl"
/>

Die Bedingung wird wahr, wenn die ausgewählte Versandart dem eingegebenen Wert entspricht, basierend auf einer der folgenden Unterbedingungen:

- **ist** (Trifft zu, wenn die Lieferauswahl genau dem eingegebenen Wert entspricht)
- **ist nicht** (Trifft zu, wenn die Lieferauswahl vom eingegebenen Wert abweicht)

Für eine der oben genannten Optionen wählen Sie eine Versandart aus der Liste: **Abholung**, **Lokale Lieferung**, **Versand**, **Abholstation**, **Einzelhandel** oder **Keine**.

> **Beispiel:** `ist nicht: Abholung` → Trifft zu, wenn die ausgewählte Versandart etwas anderes als „Abholung“ ist, d.h. „Versand“.
