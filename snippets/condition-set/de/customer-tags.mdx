### Kundentags

Diese Bedingung prüft die **Tags des Kunden** anhand bestimmter Kriterien, basierend auf einer der folgenden Unterbedingungen:

- **Hat mindestens einen dieser Werte** (Trifft zu, wenn der Kunde mindestens einen der eingegebenen Tags hat)
- **Hat keinen dieser Werte** (Trifft zu, wenn der Kunde keinen der eingegebenen Tags hat)

<Warning>
  Sie können diese Bedingung nur mit denselben Werten im gesamten Regelblock verwenden. 
  Wenn Sie unterschiedliche Werte in verschiedenen Regelblöcken verwenden möchten, sollten 
  Sie zu Bedingungsset 5 wechseln, das mehr Flexibilität bietet.
</Warning>

Um diese Bedingungen festzulegen, geben Sie einen oder mehrere Tags in das Feld ein und wählen Sie den entsprechenden Vergleich aus dem Dropdown-Menü. Trennen Sie mehrere Tags durch Kommas und drücken Sie dann die Eingabetaste, um die Tags gleichzeitig zu laden.

> Zum Beispiel, `Hat mindestens einen dieser Werte: VIP, VVIP` → Trifft auf Kunden zu, die entweder den „VIP“- oder den „VVIP“-Tag haben.
