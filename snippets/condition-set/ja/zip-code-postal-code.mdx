### 郵便番号

この条件は、チェックアウトページの配送先住所にある郵便番号フィールドの値を、1つ以上の特定の郵便番号と比較するために使用されます。以下のサブ条件に基づきます：

- **含むいずれかの値**（フィールドがリストのいずれかの値を含む場合に一致）
- **含まないいずれの値もなし**（フィールドがリストの値を含まない場合に一致）
- **範囲内の値を含む**（フィールドの値が指定範囲内にある場合に一致）
- **範囲内の値を含まない**（フィールドの値が指定範囲外にある場合に一致）
- **リスト以外の値を含む**（フィールドがリスト以外の値を含む場合に一致）
- **リスト以外の値を含まない**（フィールドがリストの値のみで構成される場合に一致）
- **数字のみを含む**（フィールドが0–9の数字のみで構成される場合に一致）
- **数字のみではない**（フィールドが数字以外の文字、スペース、記号を含む場合に一致）
- **桁数が範囲内**（フィールドの数字の桁数が指定範囲内の場合に一致）
- **桁数が範囲外**（フィールドの数字の桁数が指定範囲外の場合に一致）
- **空である**（フィールドが空の場合に一致）
- **空でない**（フィールドに何らかの値がある場合に一致）

**含むいずれかの値**、**含まないいずれの値もなし**、**リスト以外の値を含む**、**リスト以外の値を含まない** を使用して、特定の郵便番号に対して一致させます。複数の値はカンマで区切ります。

<Note>
  * ワイルドカード `*` を使って複数の郵便番号にマッチできます。例えば、`BT*` は `BT12`、`bt-590` などに一致します。  
  * 単一文字のマッチには `?` を使用します。
</Note>

> 例: `含むいずれかの値: BT*` → 「BT」で始まるすべての郵便番号に一致。

  alt="郵便番号に含む"
/>

> 例: `含まないいずれの値もなし: 12345, 67890` → フィールドが「12345」または「67890」でない場合に一致。

<img
  src="/images/essentials/condition-sets/dont-have-zipcodes.webp"
  alt="郵便番号に含まない"
/>

**範囲内の値を含む / 範囲内の値を含まない** を使用して、数値範囲内または範囲外の郵便番号をマッチさせます。

<Note>
  数字のみを入力してください。指定された開始値から終了値までの範囲内のすべての値が含まれます。
</Note>

> 例: `範囲内の値を含む: 10000 – 20000` → 10000から20000までのコードに一致。

<img
  src="/images/essentials/condition-sets/in-between-value-zipcodes.webp"
  alt="範囲内の郵便番号"
/>

**数字のみ / 数字以外** を使用して、0–9のみのコード、または文字・記号・スペースを含むコードを判定できます。

> 例: `数字のみ` → 数字のみ (0–9) に一致。

<img
  src="/images/essentials/condition-sets/zipcode-no-field-digits.webp"
  alt="数字のみの郵便番号"
/>

**桁数が範囲内 / 範囲外** を使用して、郵便番号の数字桁数に基づいて判定できます。

<Note>
  数字のみを入力してください。指定範囲内のすべての桁数が対象となります。
</Note>

> 例: `桁数が範囲内: 4 – 6` → 4桁以上6桁以下の郵便番号に一致。

<img
  src="/images/essentials/condition-sets/in-between-digits-zipcodes.webp"
  alt="桁数範囲内の郵便番号"
/>

**空である / 空でない** を使用して、フィールドが入力されているかどうかだけを確認できます。

> 例: `空でない` → フィールドに値が入力されている場合に一致。

<img
  src="/images/essentials/condition-sets/zipcode-no-field.webp"
  alt="郵便番号フィールド空でない"
/>
