### 県コード / 州コード

この条件は、チェックアウトページの配送先住所にある県/州フィールドの値を、1つ以上の特定の値または状態と比較するために使用されます。以下のサブ条件に基づきます：

- **含むいずれかの値**（県フィールドがリスト内の少なくとも1つの値を含む場合に一致）
- **含まないいずれの値もなし**（県フィールドがリスト内の値を含まない場合に一致）
- **リスト以外の値を含む**（県フィールドがリスト外の値を含む場合に一致）
- **リスト以外の値を含まない**（県フィールドがリスト内の値のみで構成される場合に一致）
- **空である**（県フィールドに値がない場合に一致）
- **空でない**（県フィールドに値がある場合に一致）

**含むいずれかの値**、**含まないいずれの値もなし**、**リスト以外の値を含む**、**リスト以外の値を含まない** の場合は、1つ以上の値を入力してください。複数の値はカンマで区切ります。

<Note>
  2文字または3文字の県/州コード（ISO_3166-2形式）を入力してください。
</Note>

> 例: `リスト以外の値を含まない: CA, TX` → フィールドが「CA」または「TX」のみで構成され、その他の値を含まない場合に一致。

**空である** または **空でない** の場合は、値の入力は不要です。

> 例: `空でない` → フィールドに値が入力されている場合に一致。
