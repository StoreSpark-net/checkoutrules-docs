### フルネーム（名 + 姓）

この条件は、顧客のフルネームを特定の値、文字数、または状態と比較するために使用されます。サブ条件は以下の通りです：

- **含む**（フルネームが指定した値または部分的な値を含む場合に一致）
- **含まない**（フルネームが指定した値または部分的な値を含まない場合に一致）
- **空である**（名前が入力されていない場合に一致）
- **空でない**（名前フィールドに値が入力されている場合に一致）
- **等しい**（フルネームが指定した値と完全に一致する場合に一致）
- **等しくない**（フルネームが指定した値と完全に一致しない場合に一致）
- **文字数が以上**（名前の総文字数が指定した数以上の場合に一致）
- **文字数が以下**（名前の総文字数が指定した数以下の場合に一致）
- **指定文字のみ**（フルネームがドロップダウンで選択した許可された文字種のみを含む場合に一致）
- **指定文字以外を含む**（フルネームがドロップダウンで選択した文字種以外の文字を含む場合に一致）

**含む、含まない、等しい、等しくない**の場合は、比較する値を入力してください。

> 例：`含まない: Test` → 名前に「Test」が含まれていない場合に一致。

<img

/>

**指定文字のみ**および**指定文字以外を含む**の場合は、ドロップダウンから文字種を選択します：

- **英字のみ**（A–Zおよびa–zのみ許可）
- **英字 + アクセント付き文字**（A–Z、a–zに加え、é、ñ、öなどのアクセント付き文字を許可）
  > 例：`指定文字のみ: 英字のみ` → 顧客の名前が英字のみで構成されている場合に一致。

<img
  src="/images/essentials/condition-sets/full-name-has-only-english-chars.webp"
  alt="フルネーム 英字のみ"
/>

**文字数が以上**または**文字数が以下**の場合は、文字数の閾値を入力してください。

> 例：`文字数が以上 10` → フルネームが10文字以上の場合に一致。

<img
  src="/images/essentials/condition-sets/full-name-has-greater-value.webp"
  alt="フルネーム 文字数以上"
/>

**空である**および**空でない**の場合は、値の入力は不要です。

> 例：`空である` → フルネームフィールドが空の場合に一致。

<img
  src="/images/essentials/condition-sets/full-name-is-empty.webp"
  alt="フルネーム 空"
/>
