### ユーザー言語

この条件は、顧客が選択した言語を確認します。サブ条件は以下の通りです:

- **次の値のいずれかを含む**（顧客の言語が指定されたリストに含まれる場合に一致）
- **次の値のいずれも含まない**（顧客の言語が指定されたリストに含まれない場合に一致）
- **次の値以外の値を含む**（顧客の言語が指定されたリストに含まれない場合に一致）
- **次の値以外の値を含まない**（顧客の言語が指定されたリストにのみ含まれる場合に一致）

上記のいずれの場合も、リストから1つ以上の言語を検索して選択してください。**貼り付け（Paste）**, **すべてコピー（Copy all）**, **すべてクリア（Clear all）** のオプションを使用して簡単に選択することもできます。

> 例: `次の値のいずれも含まない: English, French` → フィールドに「English」または「French」が含まれない場合に一致します。
