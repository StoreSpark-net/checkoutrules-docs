### Etiqueta de producto

Esta condición verifica la **etiqueta de producto** en el pedido según criterios específicos. Se considerará verdadera si **cualquier** producto en el carrito tiene el valor de Etiqueta de Producto ingresado, basado en una de las siguientes subcondiciones:

- **tiene el valor de** (Coincide cuando la etiqueta de producto tiene el valor especificado)
- **no tiene el valor de** (Coincide cuando la etiqueta de producto no tiene el valor especificado)
- **tiene cualquier otro valor distinto de** (Coincide cuando la etiqueta de producto tiene cualquier valor distinto al ingresado)
- **no tiene ningún otro valor además de** (Coincide cuando la etiqueta de producto tiene solo el valor ingresado y ningún otro)

Para configurar estas condiciones, ingrese el valor en el campo “Ingresar valor” y luego elija la comparación apropiada del menú desplegable.

> Por ejemplo, `tiene el valor de NewArrival` → Coincide cuando la etiqueta de producto es **NewArrival**.
