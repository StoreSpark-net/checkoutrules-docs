### Mercado

Se utiliza para comparar el **valor actual del mercado** en la página de pago, con uno o más mercados específicos dados (desde la configuración de mercados de Shopify en su tienda), basado en una de las siguientes sub-condiciones:

- **Tiene alguno de estos valores** (Coincide si el mercado contiene al menos uno de los mercados dados en la lista)
- **No tiene ninguno de estos valores** (Coincide si el mercado no contiene ninguno de los mercados dados en la lista)
- **Tiene algún otro valor además de estos valores** (Coincide si el mercado contiene mercados distintos a los dados en la lista)
- **No tiene ningún otro valor además de estos valores** (Coincide si el mercado contiene solo los mercados dados en la lista y ninguno más)

Para cualquiera de las opciones anteriores, busque y seleccione uno o más mercados de la lista. También puede usar las opciones **Pegar**, **Copiar todo**, o **Borrar todo** para una selección rápida.

> Por ejemplo, `No tiene ningún otro valor además de estos valores: Pakistan` → Coincide si el campo contiene solo “Pakistan” y ningún otro mercado.
