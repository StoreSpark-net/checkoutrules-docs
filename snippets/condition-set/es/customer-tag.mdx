### Etiqueta del cliente

Esta condición verifica la **etiqueta del cliente** según criterios específicos, basado en una de las siguientes subcondiciones:

- **tiene valor de** (Coincide cuando las etiquetas del cliente incluyen el valor ingresado)
- **no tiene valor de** (Coincide cuando las etiquetas del cliente no incluyen el valor ingresado)

Para configurar estas condiciones, ingrese el valor de la etiqueta en el campo y elija la comparación apropiada del menú desplegable.

> Por ejemplo, `tiene valor de VIP` → Coincide con clientes que tengan la etiqueta **VIP**.
