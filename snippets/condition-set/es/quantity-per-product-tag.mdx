### Cantidad por etiqueta de producto

Esta condición verifica la **cantidad de productos con una etiqueta de producto específica** en el pedido contra criterios definidos, basándose en una de las siguientes subcondiciones:

- **la cantidad es mayor o igual que** (Coincide cuando la cantidad de productos con la etiqueta de producto especificada es mayor o igual al número ingresado)
- **la cantidad es menor o igual que** (Coincide cuando la cantidad de productos con la etiqueta de producto especificada es menor o igual al número ingresado)
- **la cantidad es divisible por** (Coincide cuando la cantidad de productos con la etiqueta de producto especificada es un múltiplo exacto del número ingresado)
- **la cantidad no es divisible por** (Coincide cuando la cantidad de productos con la etiqueta de producto especificada no es un múltiplo del número ingresado)

Para establecer estas condiciones:

<Steps>
  <Step title="Ingresar etiqueta de producto">
    Ingresa la <strong>etiqueta de producto</strong> en el campo “Ingresar etiqueta de producto”.
  </Step>

<Step title="Ingresar cantidad">
  Ingresa la <strong>cantidad</strong> en el campo “Ingresar cantidad”.
</Step>

  <Step title="Seleccionar opción de comparación">
    Selecciona la opción de comparación del menú desplegable.
  </Step>
</Steps>

> **Ejemplo para “la cantidad es mayor o igual que”**:  
> Etiqueta de producto: `SummerCollection`  
> Cantidad: `3`  
> → Coincide con pedidos donde hay 3 o más unidades de productos etiquetados como `SummerCollection`.
