### Marché

Utilisé pour comparer la valeur réelle du marché sur la page de paiement avec un ou plusieurs marchés spécifiques donnés (à partir de la configuration des marchés Shopify dans votre boutique), en fonction de l’une des sous-conditions suivantes :

- **Contient l’une de ces valeurs** (Correspond si le marché contient au moins l’un des marchés donnés dans la liste)
- **Ne contient aucune de ces valeurs** (Correspond si le marché ne contient aucun des marchés donnés dans la liste)
- **Contient une autre valeur que ces valeurs** (Correspond si le marché contient des marchés autres que ceux donnés dans la liste)
- **Ne contient aucune autre valeur que ces valeurs** (Correspond si le marché contient uniquement les marchés donnés dans la liste et aucun autre)

Pour l’une des options ci-dessus, recherchez et sélectionnez un ou plusieurs marchés dans la liste. Vous pouvez également utiliser les options **Coller**, **Tout copier**, ou **Tout effacer** pour une sélection rapide.

> Par exemple, `Ne contient aucune autre valeur que ces valeurs : Pakistan` → Correspond si le champ contient uniquement “Pakistan” et aucun autre marché.
