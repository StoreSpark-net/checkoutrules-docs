### Étiquette client

Cette condition vérifie l’**étiquette du client** par rapport à des critères spécifiques, en fonction de l’une des sous-conditions suivantes :

- **a pour valeur** (Correspond lorsque les étiquettes du client incluent la valeur saisie)
- **n’a pas pour valeur** (Correspond lorsque les étiquettes du client n’incluent pas la valeur saisie)

Pour définir ces conditions, saisissez la valeur de l’étiquette dans le champ et choisissez la comparaison appropriée dans le menu déroulant.

> Par exemple, `a pour valeur VIP` → Correspond aux clients qui ont l’étiquette **VIP**.
