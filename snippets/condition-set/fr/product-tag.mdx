### Étiquette de produit

Cette condition vérifie **l’étiquette de produit** dans la commande par rapport à des critères spécifiques. Elle sera considérée comme remplie si **un quelconque** produit dans le panier possède la valeur de l’étiquette de produit saisie, en fonction de l’une des sous-conditions suivantes :

- **a la valeur de** (Correspond lorsque l’étiquette de produit a la valeur spécifiée)
- **n’a pas la valeur de** (Correspond lorsque l’étiquette de produit n’a pas la valeur spécifiée)
- **a toute autre valeur que** (Correspond lorsque l’étiquette de produit a une valeur différente de celle saisie)
- **n’a aucune autre valeur que** (Correspond lorsque l’étiquette de produit n’a que la valeur saisie et aucune autre)

Pour définir ces conditions, saisissez la valeur dans le champ “Saisir la valeur”, puis choisissez la comparaison appropriée dans le menu déroulant.

> Par exemple, `a la valeur de NewArrival` → Correspond lorsque l’étiquette de produit est **NewArrival**.
