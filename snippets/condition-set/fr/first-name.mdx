### Prénom

Utilisé pour comparer le prénom du client avec une ou plusieurs valeurs spécifiques, des nombres ou des états, selon l’une des sous-conditions suivantes :

- **Contient** (Correspond si le prénom contient la valeur spécifiée ou une valeur partielle)
- **Ne contient pas** (Correspond si le prénom ne contient pas la valeur spécifiée ou une valeur partielle)
- **Est vide** (Correspond si aucun prénom n’est saisi)
- **N’est pas vide** (Correspond si une valeur est saisie dans le champ du prénom)
- **Est** (Correspond si le prénom correspond exactement à la valeur spécifiée)
- **N’est pas** (Correspond si le prénom ne correspond pas exactement à la valeur spécifiée)
- **A un nombre de caractères supérieur ou égal à** (Correspond si le nombre total de caractères dans le prénom est supérieur ou égal au nombre spécifié)
- **A un nombre de caractères inférieur ou égal à** (Correspond si le nombre total de caractères dans le prénom est inférieur ou égal au nombre spécifié)
- **Contient uniquement** (Correspond si le prénom contient uniquement le type de caractères autorisé depuis le menu déroulant)
- **Ne contient pas uniquement** (Correspond si le prénom contient des caractères autres que le type de caractères autorisé depuis le menu déroulant)

Pour **Contient**, **Ne contient pas**, **Est** et **N’est pas**, saisissez la valeur à comparer.

> Par exemple, `N’est pas : Jane` → Correspond si le prénom est différent de “Jane”.

Pour **Contient uniquement** et **Ne contient pas uniquement**, sélectionnez le type de caractères dans le menu déroulant :

- **Caractères anglais** (Autorise uniquement A–Z et a–z)
- **Caractères anglais + variantes accentuées** (Autorise A–Z, a–z, plus les caractères accentués comme é, ñ, ö)  
  > Par exemple, `contient uniquement : caractères anglais` → Correspond si le prénom contient uniquement des caractères anglais.

<img
  src="/images/essentials/condition-sets/first-name-has-only.webp"
  alt="Le prénom contient uniquement"
/>

Pour **A un nombre de caractères supérieur ou égal à** ou **A un nombre de caractères inférieur ou égal à**, saisissez le seuil de caractères.

> Par exemple, `A un nombre de caractères supérieur ou égal à 5` → Correspond si le prénom contient 5 caractères ou plus.

<img
  src="/images/essentials/condition-sets/first-name-has-greater-value.webp"
  alt="Le prénom a une valeur supérieure"
/>

Pour **Est vide** et **N’est pas vide**, aucune saisie de valeur n’est nécessaire.

> Par exemple, `Est vide` → Correspond si le champ du prénom est laissé vide.

<img
  src="/images/essentials/condition-sets/first-name-is-empty.webp"
  alt="Le prénom est vide"
/>
