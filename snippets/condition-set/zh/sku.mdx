### SKU

此条件用于检查购物车中产品的 **SKU** 是否符合特定条件。如果购物车中 **任意** 产品的 SKU 符合输入值，则条件成立，基于以下子条件之一：

- **包含任意这些值**（当 SKU 至少包含列表中的一个值时匹配）
- **不包含任何这些值**（当 SKU 不包含列表中的任何值时匹配）
- **包含除这些值之外的任何值**（当 SKU 包含列表之外的值时匹配）
- **不包含除这些值之外的任何值**（当 SKU 仅包含输入值且不包含其他值时匹配）
- **值的数量大于或等于**（当 SKU 数量大于或等于输入值时匹配）
- **值的数量小于或等于**（当 SKU 数量小于或等于输入值时匹配）

对于 **包含任意这些值**、**不包含任何这些值**、**包含除这些值之外的任何值** 和 **不包含除这些值之外的任何值**，在字段中输入一个或多个 SKU 值，多个值用逗号分隔。

<Note>
  使用 * 通配符匹配多个 SKU 值。例如，SKU* 将匹配 SKU12、sku-new-1 等。
</Note>

> 例如，`包含任意这些值: SKU1, SKU2` → 如果 SKU 是 “SKU1” 或 “SKU2”，则匹配。

对于 **值的数量大于或等于** 或 **值的数量小于或等于**，输入单个数字值以匹配 SKU 数量。

> 例如，`值的数量大于或等于 3` → 如果 SKU 数量为三个或更多，则匹配。
