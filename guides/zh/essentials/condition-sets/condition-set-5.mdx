## B2B 详情

### 采购公司元字段

此条件会检查您指定的 **采购公司自定义字段**（通过 **命名空间** 和 **键**），然后使用以下子条件之一进行评估：

- **具有以下任意值**（如果自定义字段至少有一个输入的值，则匹配）
- **不具有以下任意值**（如果自定义字段没有输入的任何值，则匹配）
- **具有除以下值以外的其他值**（如果自定义字段的值不在输入值中，则匹配）
- **不具有除以下值以外的其他值**（如果自定义字段只有输入的值而没有其他值，则匹配）
- **为空值**（如果自定义字段没有值，则匹配）
- **不为空值**（如果自定义字段有任意值，则匹配）
- **数值大于或等于**（如果自定义字段的数值大于或等于输入的数字，则匹配）
- **数值小于或等于**（如果自定义字段的数值小于或等于输入的数字，则匹配）
- **值的数量大于或等于**（如果自定义字段中值的数量大于或等于输入的数字，则匹配）
- **值的数量小于或等于**（如果自定义字段中值的数量小于或等于输入的数字，则匹配）

对于 **具有以下任意值**、**不具有以下任意值**、**具有除以下值以外的其他值** 和 **不具有除以下值以外的其他值**，请输入一个或多个值到字段中。多个值用逗号分隔。

> 例如，`命名空间: profile` · `键: industry` · `具有以下任意值: Retail, Wholesale` → 如果采购公司的自定义字段包含“Retail”或“Wholesale”，则匹配。

<img
  src="/images/essentials/condition-sets/purchasing-company-metafield-has-any-value.webp"
  alt="采购公司自定义字段具有任意值"
/>

对于 **数值大于或等于** 和 **数值小于或等于**，请输入单个数值以与自定义字段的数值进行匹配。

> 例如，`命名空间: finance` · `键: annual_revenue` · `数值大于或等于 1000000` → 如果采购公司的年收入大于等于一百万，则匹配。

<img
  src="/images/essentials/condition-sets/purchasing-company-metafield-greater-value.webp"
  alt="采购公司自定义字段数值比较"
/>

对于 **值的数量大于或等于** 和 **值的数量小于或等于**，请输入单个数值以与自定义字段中的值数量进行匹配。

> 例如，`命名空间: compliance` · `键: certifications` · `值的数量小于或等于 2` → 如果公司有两个或更少的认证，则匹配。

<img
  src="/images/essentials/condition-sets/purchasing-company-metafield-values-count.webp"
  alt="采购公司自定义字段数量"
/>

### 产品变型选项

此条件会检查 **产品变体选项** 是否符合特定条件。如果购物车中的任何产品具有输入的产品变体选项值，将根据以下子条件之一进行匹配：

- **具有以下任意值**（如果产品变体选项至少有一个输入的值，则匹配）
- **不具有以下任意值**（如果产品变体选项没有输入的任何值，则匹配）
- **具有除以下值以外的其他值**（如果产品变体选项的值不在输入值中，则匹配）
- **不具有除以下值以外的其他值**（如果产品变体选项只有输入的值而没有其他值，则匹配）
- **值的数量大于或等于**（如果产品变体选项的数量大于或等于输入的数字，则匹配）
- **值的数量小于或等于**（如果产品变体选项的数量小于或等于输入的数字，则匹配）

对于 **具有以下任意值**、**不具有以下任意值**、**具有除以下值以外的其他值** 和 **不具有除以下值以外的其他值**，请输入一个或多个产品变体选项值。多个值用逗号分隔。

> 例如，`具有以下任意值: Red, Blue` → 如果产品变体选项包含“Red”或“Blue”，则匹配。

对于 **值的数量大于或等于** 和 **值的数量小于或等于**，请输入单个数值以与产品变体选项数量进行匹配。

> 例如，`值的数量大于或等于 3` → 如果有 3 个或更多产品变体选项，则匹配。
