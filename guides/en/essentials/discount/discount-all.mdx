## Rule overview

This is the primary discount rule in the current app flow. It lets you combine **order**, **product**, and **shipping** discounts in one rule, each with its own operation options.

For each block, you choose conditions and then enable one or more discount types (order/product/shipping). The app applies eligible discounts according to your selected strategy settings.

## Supported operations

### Order discount operations

- Percentage
- Fixed amount off
- Percentage excluding these products
- Percentage for only these products
- Fixed amount off excluding these products
- Fixed amount off for only these products
- Fixed price subtotal percentage

### Product discount operations

- Percentage
- Fixed amount off
- Set fixed amount
- Set all products free
- Fixed price subtotal percentage

### Shipping discount operations

- Percentage
- Fixed amount off
- Set fixed price
- Set all shipping free
- Subtotal percentage
- Fixed price subtotal percentage

## Sample configuration

### Create a new rule

From your [Shopify admin](/guides/en/get-started/installation#go-to-the-app), open the app. In the sidebar, go to [Rules](/guides/en/get-started/quickstart#create-a-sample-rule) and click _"Create Rule"_.

Then choose:

- category **Discount**
- checkout rule type **Discount all**
- click **Select Rule**

Choose **Automatic discount** or **Discount code**, then set [status](/guides/en/get-started/concepts#status).

### Configure shipping discount action

Choose a shipping operation, set value/message where applicable, and set method scope.

<img
  src="/images/essentials/shipping/action-type-discount-shipping-methods.webp"
  alt="Shipping Action Type"

<img

If shipping method scope is **only these shipping methods**:

- Supported operators are **contains**, **does not contain**, **is**, and **is not**
- Matching is case-insensitive
- `*` wildcard is supported
- Multiple method rows are OR logic

<Warning>
  Shopify may show USPS-prefixed names at checkout. For method matching, remove
  the USPS prefix in your input. Example: use `Ground Advantage` instead of
  `USPS Ground Advantage`.
</Warning>

Subscription deliveries can be configured when shipping discount is enabled:

- Limit discount to first delivery
- First delivery discounted + recurring deliveries free (limited)
- First delivery discounted + all recurring deliveries free

### Fallback behavior for Date Time / GeoIP conditions

If any block uses **Date Time** or **Customer location (using GeoIP)** conditions:

- **Do nothing**: matching action is skipped when time/date or location context is unavailable.
- **Do this action**: fallback action runs instead.

## Limitations

- This rule is incompatible with draft-order-based checkout and apps that use draft-order checkout.
- If checkout URL contains `.../checkouts/do/...`, `do` means draft order and this rule will not run.
