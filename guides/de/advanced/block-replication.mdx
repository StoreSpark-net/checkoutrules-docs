## Übersicht
Die Block-Replikation ermöglicht es Ihnen, bestehende Logik in einer Regel schnell zu duplizieren, ohne sie manuell neu erstellen zu müssen. Dies ist besonders nützlich, wenn der Inhalt der Blöcke ähnlich ist, da es Zeit spart und Konsistenz gewährleistet.

## Verwendung
### Block kopieren

Klicken Sie auf das **Block kopieren**-Symbol bei dem Block, den Sie duplizieren möchten. Dadurch wird der gesamte Block kopiert, einschließlich seiner Bedingungen und Aktionen.

<Note>
Nach dem Kopieren erscheinen das **Häkchen-Symbol** und der Text **"Kopiert"** anstelle des Block kopieren-Symbols. Außerdem wird am unteren Bildschirmrand eine Bestätigungsmeldung **"Block erfolgreich kopiert!"** angezeigt.
</Note>

<img
  src="/images/advanced/block-replication/copy-block.webp"
  alt="Block kopieren"
/>

### Block einfügen
Fügen Sie den kopierten Block ein, indem Sie auf das **Block einfügen**-Symbol innerhalb derselben Regel oder einer anderen Regel desselben Typs klicken.

<Note>
Eine Bestätigungsmeldung **"Block erfolgreich eingefügt!"** wird am unteren Bildschirmrand angezeigt.
</Note>

<img
  src="/images/advanced/block-replication/paste-block.webp"
  alt="Block einfügen"
/>

<Warning>
Wenn Sie mehr als einen Block kopieren, wird nur der **zuletzt kopierte Block** eingefügt, nicht alle.
</Warning>

### Inhalt aktualisieren
Passen Sie die notwendigen Bedingungen, Werte oder Aktionen im eingefügten Block an, um dem neuen Szenario zu entsprechen. So können Sie die Struktur wiederverwenden und gleichzeitig für den neuen Fall anpassen.

**Beispiel:**  
Wenn Sie einen Block kopiert haben, bei dem die Unterbedingung **Benutzersprache** auf **Französisch** eingestellt ist und die Fehlermeldung auf Französisch angezeigt wird, können Sie nach dem Einfügen Folgendes tun:  
- Ändern Sie die Unterbedingung **Benutzersprache** von **Französisch** auf **Deutsch**.  
- Ersetzen Sie den Fehlermeldungstext durch die deutsche Übersetzung.  

Auf diese Weise bleibt die Logik des Blocks gleich (Mindestbestellwertprüfung), aber die Details werden für eine andere Sprache angepasst.

  src="/images/advanced/block-replication/update-pasted-content.webp"

/>

## Beispielanwendung

Erstellen wir eine Regel, die einen **Mindestbestellwert** erzwingt und Fehlermeldungen in mehreren Sprachen anzeigt.  

Durch die Verwendung von **Block-Replikation** müssen Sie nur den ersten Block (Französisch) erstellen, dann einfach kopieren und innerhalb derselben Regel einfügen und die **Benutzersprache**-Bedingung sowie die Fehlermeldung für Deutsch aktualisieren. Schließlich fügen Sie einen Zurückgreifen Block für Englisch hinzu.  

### Block 1 (Französisch)
- Fügen Sie die Bedingung **Bestellsumme** hinzu und setzen Sie sie auf **kleiner oder gleich 99,99**, um einen Fehler auszulösen, falls der Bestellbetrag unter 100 liegt.  
- UND Unterbedingung **Benutzersprache** = **Französisch**.  
- Fehlermeldung setzen:  
  > "Le montant de la commande est inférieur au montant minimum requis de 100 $"

<img
  src="/images/advanced/translations/example-french-translation.webp"
  alt="Beispiel Französische Übersetzung"
/>

### Block 2 (Deutsch) — erstellt durch Kopieren von Block 1
- Fügen Sie den kopierten Block unter allen Blöcken ein, indem Sie auf das **Block einfügen**-Symbol innerhalb derselben Regel klicken.  
- Aktualisieren Sie die Unterbedingung **Benutzersprache** auf **Deutsch**.  
- Fehlermeldung aktualisieren:  
  > "Der Bestellbetrag liegt unter dem erforderlichen Mindestbestellwert von 100 $"

<img
  src="/images/advanced/translations/example-german-translation.webp"
  alt="Beispiel Deutsche Übersetzung"
/>

### Block 3 (Standard Zurückgreifen – Englisch)
- Fügen Sie die Bedingung **Bestellsumme** hinzu und setzen Sie sie auf **kleiner oder gleich 99,99**, um einen Fehler auszulösen, falls der Bestellbetrag unter 100 liegt.  
- Für die **Else / Zurückgreifen Bedingung**, UND Unterbedingung **Alle vorherigen Bedingungen ungültig**.  
- Fehlermeldung setzen:  
  > "Order amount is less than the required minimum order amount of $100"

<img
  src="/images/advanced/translations/example-fallback-english-translation.webp"
  alt="Beispiel Zurückgreifen Englische Übersetzung"
/>
