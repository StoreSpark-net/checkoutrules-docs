## Übersicht

Das Duplizieren von Regeln ermöglicht es Ihnen, schnell eine neue Regel auf Basis einer bestehenden zu erstellen.  
Anstatt eine Regel von Grund auf neu zu erstellen, können Sie sie duplizieren, einige Änderungen vornehmen und in kürzerer Zeit einrichten.  

Dies ist besonders nützlich, wenn die neue Regel, die Sie erstellen möchten, der bestehenden sehr ähnlich ist.

## Verwendung
### Regeln auswählen  
Sie können eine oder mehrere Regeln auswählen, die Sie duplizieren möchten.  

<img
  src="/images/advanced/rule-duplication/select-rules.webp"
  alt="Regeln auswählen"
/>

### Duplizieren-Aktion 
Klicken Sie auf die drei Punkte und dann auf **"Duplizieren"**. Ein Bestätigungs-Popup wird angezeigt.

<img
  src="/images/advanced/rule-duplication/duplicate-action.webp"
  alt="Duplizieren Aktion"
/>

Klicken Sie dann im Popup auf **"Duplizieren"**, um Kopien der ausgewählten Regeln zu erstellen.

<img
  src="/images/advanced/rule-duplication/popup-duplicate-copies.webp"
  alt="Popup Duplizierte Kopien"
/>

### Regeln bearbeiten  
Aktualisieren Sie die duplizierten Regeln, indem Sie nur die Teile ändern, die angepasst werden müssen, wie Bedingungen oder Werte.  

> Zum Beispiel: Wenn die Originalregel einen **10% Rabatt** auf Bestellungen über **100 Dollar** gewährt, können Sie sie duplizieren und die Bedingung ändern, damit der Rabatt auf Bestellungen über **150 Dollar** angewendet wird.

<img
  src="/images/advanced/rule-duplication/modify-rules.webp"
  alt="Regeln bearbeiten"
/>

### Regeln speichern  
Nachdem die Änderungen vorgenommen wurden, speichern Sie die neuen Regeln, ohne die Originalregeln zu beeinflussen.  

<img
  src="/images/advanced/rule-duplication/save-rules.webp"
  alt="Regeln speichern"
/>

## Beispielanwendung  
Angenommen, Sie haben bereits eine Regel, die einen Rabatt für Kunden in den **Vereinigten Staaten** gewährt. Wenn Sie dieselbe Regel für Kunden in **Kanada** erstellen möchten, können Sie dies in zwei einfachen Schritten tun:

- Erstellen Sie eine Kopie der bestehenden **US-Rabattregel**. 

<img
  src="/images/advanced/rule-duplication/duplicate-united-states-rule.webp"
  alt="US-Regel duplizieren"
/>

- Aktualisieren Sie die Standort-Einstellung auf **Kanada** und speichern Sie die neue Regel.  

<img
  src="/images/advanced/rule-duplication/change-country-in-duplicated-rule.webp"
  alt="Land in duplizierter Regel ändern"
/>
